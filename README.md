# ChatBot-Implementation-using-Chainlit



# how to run?

1. Create a virtual environment 

```python -m venv myenv```

2. Active the environment 

```llmapp\Scripts\activate```

3. Install the require packages 

```pip install -r requirement.txt```


# chainlit commands  

```chainlit init```

# To run the app.py file
```chainlit run app.py```
